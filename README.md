# 💌 Love Letter for Noaa

A romantic, interactive love letter built with React and Framer Motion.

## 🌟 Features

- **6 Beautiful Pages** with smooth animations
- **Responsive Design** - works on all devices
- **Interactive Elements** - polaroid photos gallery
- **Handwriting Fonts** - elegant Caveat font styling
- **Animated Effects** - stars, particles, floating hearts
- **Glassmorphism Design** - modern UI with backdrop filters

## 📖 Pages

1. **Page 1** - Opening with stars and title animation
2. **Page 2** - Heartfelt message on paper texture
3. **Page 3** - Time-based reflection with clock animation
4. **Page 4** - Personal love letter
5. **Page 5** - Photo gallery with polaroid style
6. **Page 6** - Grand finale with hearts animation

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/ntnknpw9-art/jnikn.git

# Navigate to the directory
cd jnikn

# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## 🛠️ Build

```bash
npm run build
```

Creates an optimized production build.

## 📦 Dependencies

## 🌐 Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Select your GitHub repository
5. Click "Deploy"

The app will be live in seconds!

### Deploy to Netlify

1. Run `npm run build`
2. Drag and drop the `build` folder to [Netlify](https://app.netlify.com)
